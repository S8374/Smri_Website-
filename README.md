🛒 Smri Shop - E-Commerce Platform
Smri Shop is a modern and feature-rich e-commerce platform that allows users to browse, purchase, and manage products efficiently. Built with React, Tailwind CSS, Firebase, and MongoDB, this platform ensures a seamless shopping experience.

🚀 Features
🔹 User Features:
✅ Authentication & Authorization – Secure login and registration using Firebase.
✅ Product Listings – View a wide range of products with detailed information.
✅ Cart Management – Add, update, and remove products from the cart.
✅ Order Tracking – Users can track their order status in real-time.
✅ Payment Integration – Secure payment system using Stripe.

🔹 Admin Features:
✅ Dashboard Analytics – View sales reports, user activities, and revenue insights.
✅ Product Management – Add, edit, and delete products from the catalog.
✅ Order Management – Update order statuses and process refunds.

🔹 Seller Features:
✅ Product Listings for Sellers – Sellers can upload and manage their own products.
✅ Sales Performance – View sales insights and earnings.

🛠️ Tech Stack
Technology	Purpose
React.js	Frontend UI Development
Tailwind CSS	Styling & Responsive Design
Firebase	Authentication & Hosting
MongoDB	NoSQL Database for Products & Orders
Express.js	Backend API Development
Node.js	Server-side Logic
Stripe API	Payment Processing
